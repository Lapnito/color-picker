## 什么是 Color Picker: 相机 HEX RGB？

**Color Picker: 相机 HEX RGB** 是由 [lapnito.cz s.r.o.](https://lapnito.cz) 开发的免费颜色识别应用，让您可以使用手机相机或相册中的任何照片捕获现实世界中的精确颜色。将十字准线对准任何表面，应用程序会立即识别像素颜色并提供 HEX、RGB、HSL 和 CMYK 格式的值。

在 [Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker) 上以 **Color Picker: Camera HEX RGB** 名称提供，在 [App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237) 上以 **Color Picker Camera: HEX RGB** 名称提供。

无论您是从实物样本匹配品牌颜色的平面设计师，为 CSS 提取精确颜色代码的开发者，对墙面涂料进行取样的室内装饰师，还是构建情绪板的爱好者 — Color Picker 都能通过一键点击为您提供精确、即可复制的颜色值。所有处理都在您的设备上本地进行，零数据收集、无广告。

## 主要功能

### 颜色识别

- **实时相机取色器** — 将精确的十字准线对准任何现实世界的表面，实时捕获确切的像素颜色。就像物理世界的滴管工具。
- **照片取色器** — 从相册中的任何图像提取颜色。点击照片上的任何位置即可识别其颜色。
- **多种颜色格式** — 每个捕获的颜色同时以 HEX、RGB、HSL 和 CMYK 显示，因此您始终拥有所需的格式。
- **一键复制** — 点击任何颜色代码即可立即复制到剪贴板，准备好粘贴到您的设计工具或代码编辑器中。

### 颜色组织

- **颜色库和色样** — 保存您喜爱的颜色，并在个人库中组织起来以便快速参考。
- **调色板构建器** — 手动创建自定义调色板或从取样颜色构建。将相关颜色组合成一致的调色板。
- **共享和导出** — 以文本、JSON、CSS 变量或 PNG 图像形式共享调色板。导出完整库作为备份或交付给团队成员。

### 颜色工具

- **对比度检查器** — 验证两种颜色是否满足文本可读性的对比度无障碍要求。
- **颜色和谐建议** — 从任何选定颜色生成互补、三元、类似和其他和谐色彩组合。
- **RAL 颜色匹配** — 为任何捕获的颜色找到最接近的 RAL 涂料参考 — 对绘画、工业设计和制造很有用。

### 用户体验

- **简洁直观的界面** — 极简设计专注于快速颜色捕获以及库、调色板和工具之间的轻松导航。
- **离线工作** — 无需互联网连接。所有颜色处理完全在您的设备上进行。
- **深色模式支持** — 完整的浅色和深色主题支持，适合在任何环境中舒适使用。
- **备份和恢复** — 通过"文件"应用以 JSON 格式导出和导入您的整个颜色库和调色板。

## 相机取色如何工作

Color Picker 使用手机后置相机捕获实时视频流。屏幕中央定位着一个精密的十字准线覆盖层，标记正在采样的精确像素。当您移动相机时，应用程序实时读取十字准线下像素的颜色值，并同时将其转换为多种颜色格式（HEX、RGB、HSL、CMYK）。

对于基于照片的取色，应用程序加载选定的图像并允许您点击任何像素。点击坐标处的颜色从图像的像素数据中提取并转换为所有支持的格式。这适用于您设备照片库支持的任何图像格式。

所有颜色处理都在您的设备上使用相机硬件和 GPU 本地执行。不会将任何图像或颜色数据发送到任何服务器。应用程序仅需要相机权限用于实时取色，仅需要照片库访问权限用于基于相册的颜色提取。

**相机取色器有多准确？** 准确性取决于您的相机传感器和照明条件。为获得最佳结果，请确保良好、中性的照明并保持设备稳定。应用程序从相机流中捕获精确的像素颜色，因此准确性仅受相机硬件本身的限制。

**我可以从任何照片中选取颜色吗？** 是的，您可以从设备上存储的任何图像中选取颜色 — 照片、截图、下载的图像和设计稿都可以。

## 常见问题

**问：Color Picker 免费吗？**
答：是的，该应用在 Android 和 iOS 上完全免费下载和使用。没有广告，没有应用内购买。

**问：它可以在没有互联网的情况下工作吗？**
答：是的，Color Picker 完全离线工作。所有颜色检测和处理都在您的设备上本地进行。不需要互联网连接。

**问：它支持哪些颜色格式？**
答：应用程序支持 HEX（例如 #FF5733）、RGB（例如 255、87、51）、HSL（例如 11°、100%、60%）和 CMYK（例如 0%、66%、80%、0%）。所有格式同时显示为每个选取的颜色。

**问：我可以用它作为滴管工具吗？**
答：是的，Color Picker 的工作方式与桌面设计软件的滴管工具完全相同，但适用于现实世界。将相机对准任何表面，十字准线即可识别精确的像素颜色。

**问：它会收集我的数据吗？**
答：不。Color Picker 不收集任何用户数据。您的所有颜色、调色板和设置仅存储在您的设备上。没有分析、没有跟踪、没有广告。

**问：我可以导出我的调色板以在设计工具中使用吗？**
答：是的，您可以将调色板导出为 JSON、CSS 变量、文本描述或 PNG 图像。这样可以轻松与团队成员共享颜色或将其导入 Figma、Sketch、VS Code 或任何其他工具。

**问：RAL 匹配是什么？**
答：RAL 是主要用于绘画、涂料和工业设计的标准化颜色系统。Color Picker 为任何选取的颜色找到最接近的 RAL 颜色参考，以便您识别匹配的涂料或涂层。

**问：支持哪些设备？**
答：Android 5.0 (Lollipop) 及以上，iOS 13.0 及以上。该应用还可在 iPad、iPod touch、Apple Silicon Mac 和 Apple Vision 上运行。

**问：Android 和 iOS 版本之间有什么区别？**
答：两个版本提供相同的核心功能。Android 版本名为 "Color Picker: Camera HEX RGB"，iOS 版本名为 "Color Picker Camera: HEX RGB"。
