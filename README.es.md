## ¿Qué es Color Picker: Cámara HEX RGB?

**Color Picker: Cámara HEX RGB** es una aplicación gratuita de identificación de colores de [lapnito.cz s.r.o.](https://lapnito.cz) que te permite capturar colores exactos del mundo real usando la cámara de tu teléfono o cualquier foto de tu galería. Apunta el punto de mira a cualquier superficie, y la app identifica al instante el color del píxel y proporciona su valor en formatos HEX, RGB, HSL y CMYK.

Disponible como **Color Picker: Camera HEX RGB** en [Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker) y como **Color Picker Camera: HEX RGB** en la [App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237).

Ya seas diseñador gráfico que busca igualar un color de marca a partir de una muestra física, desarrollador extrayendo códigos de color para CSS, decorador de interiores probando pinturas de pared, o aficionado creando un moodboard — Color Picker te da valores de color precisos y listos para copiar con un solo toque. Todo el procesamiento ocurre localmente en tu dispositivo, sin recolección de datos ni anuncios.

## Características principales

### Identificación de colores

- **Selector de color con cámara en vivo** — Apunta el punto de mira preciso a cualquier superficie real para capturar el color exacto del píxel en tiempo real. Funciona como un cuentagotas para el mundo físico.
- **Selector desde foto** — Extrae colores de cualquier imagen de tu galería. Toca cualquier parte de la foto para identificar su color.
- **Múltiples formatos** — Cada color capturado se muestra simultáneamente en HEX, RGB, HSL y CMYK, así siempre tienes el formato que necesitas.
- **Copia con un toque** — Toca cualquier código de color para copiarlo al portapapeles, listo para pegar en tu herramienta de diseño o editor de código.

### Organización de colores

- **Biblioteca y muestras** — Guarda tus colores favoritos y organízalos en una biblioteca personal para referencia rápida.
- **Constructor de paletas** — Crea paletas de colores personalizadas manualmente o constrúyelas a partir de colores capturados. Combina colores relacionados en paletas coherentes.
- **Compartir y exportar** — Comparte paletas como texto, JSON, variables CSS o imágenes PNG. Exporta tu biblioteca completa para respaldo o para compartir con tu equipo.

### Herramientas de color

- **Verificador de contraste** — Comprueba si dos colores cumplen los requisitos de contraste de accesibilidad para la legibilidad del texto.
- **Armonías de color** — Genera combinaciones complementarias, triádicas, análogas y otras armonías desde cualquier color capturado.
- **Coincidencia RAL** — Encuentra la referencia de pintura RAL más cercana para cualquier color capturado — útil para pintura, diseño industrial y manufactura.

### Experiencia de usuario

- **Interfaz limpia e intuitiva** con diseño minimalista centrado en la captura rápida y navegación fácil entre biblioteca, paletas y herramientas.
- **Funciona sin conexión** — No requiere internet. Todo el procesamiento de color ocurre en tu dispositivo.
- **Modo oscuro** — Soporte completo para temas claro y oscuro para uso cómodo en cualquier ambiente.
- **Copia de seguridad y restauración** — Exporta e importa tu biblioteca completa y paletas como JSON mediante la app Archivos.

## Cómo funciona la selección de color con cámara

Color Picker usa la cámara trasera del teléfono para capturar un flujo de video en vivo. Un punto de mira de precisión se coloca en el centro de la pantalla, marcando el píxel exacto que se está tomando como muestra. Al mover la cámara, la app lee el valor del píxel bajo el punto de mira en tiempo real y lo convierte simultáneamente a múltiples formatos (HEX, RGB, HSL, CMYK).

Para la selección desde foto, la app carga la imagen seleccionada y permite tocar cualquier píxel. El color en las coordenadas del toque se extrae de los datos de la imagen y se convierte a todos los formatos soportados. Funciona con cualquier formato soportado por la galería del dispositivo.

Todo el procesamiento se realiza localmente en tu dispositivo usando el hardware de cámara y la GPU. No se envían imágenes ni datos de color a ningún servidor. La app solo requiere permiso de cámara para la selección en vivo y acceso a galería para la extracción desde foto.

**¿Qué precisión tiene el selector de color con cámara?** La precisión depende del sensor de tu cámara y las condiciones de iluminación. Para los mejores resultados, asegura buena luz neutra y mantén el dispositivo estable. La app captura el color exacto del píxel desde el flujo de la cámara, así que la precisión está limitada solo por el hardware.

**¿Puedo seleccionar colores desde cualquier foto?** Sí, puedes seleccionar colores de cualquier imagen en tu dispositivo — fotos, capturas de pantalla, imágenes descargadas y mockups de diseño.

## Preguntas frecuentes

**P: ¿Color Picker es gratis?**
R: Sí, la app es completamente gratis en Android y iOS. Sin anuncios y sin compras integradas.

**P: ¿Funciona sin internet?**
R: Sí, Color Picker funciona totalmente sin conexión. Toda la detección y procesamiento de color ocurre localmente.

**P: ¿Qué formatos de color soporta?**
R: La app soporta HEX (ej. #FF5733), RGB (ej. 255, 87, 51), HSL (ej. 11°, 100%, 60%) y CMYK (ej. 0%, 66%, 80%, 0%). Todos los formatos se muestran simultáneamente para cada color capturado.

**P: ¿Puedo usarlo como cuentagotas?**
R: Sí, Color Picker funciona exactamente como un cuentagotas de un software de diseño de escritorio, pero para el mundo real. Apunta tu cámara a cualquier superficie y el punto de mira identifica el color exacto.

**P: ¿Recoge datos?**
R: No. Color Picker no recoge ningún dato de usuario. Todos tus colores, paletas y ajustes se guardan solo en tu dispositivo. Sin analíticas, sin rastreo, sin anuncios.

**P: ¿Puedo exportar mis paletas para herramientas de diseño?**
R: Sí, puedes exportar paletas como JSON, variables CSS, descripciones de texto o imágenes PNG. Así es fácil compartir colores con tu equipo o importarlos a Figma, Sketch, VS Code o cualquier herramienta.

**P: ¿Qué es la coincidencia RAL?**
R: RAL es un sistema de color estandarizado usado principalmente en pintura, recubrimientos y diseño industrial. Color Picker encuentra la referencia RAL más cercana para cualquier color capturado.

**P: ¿Qué dispositivos son compatibles?**
R: Android 5.0 (Lollipop) y superior, e iOS 13.0 y superior. La app también funciona en iPad, iPod touch, Mac con Apple Silicon y Apple Vision.

**P: ¿Cuál es la diferencia entre la versión Android y iOS?**
R: Ambas ofrecen la misma funcionalidad principal. La versión Android se llama "Color Picker: Camera HEX RGB" y la iOS "Color Picker Camera: HEX RGB".
