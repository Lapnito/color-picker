## Що таке Color Picker: Камера HEX RGB?

**Color Picker: Камера HEX RGB** — це безкоштовний застосунок для ідентифікації кольорів від [lapnito.cz s.r.o.](https://lapnito.cz), який дозволяє захоплювати точні кольори з реального світу за допомогою камери телефону або будь-якої фотографії з галереї. Наведіть перехрестя на будь-яку поверхню, і застосунок миттєво визначить колір пікселя та надасть його значення у форматах HEX, RGB, HSL і CMYK.

Доступно як **Color Picker: Camera HEX RGB** в [Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker) і як **Color Picker Camera: HEX RGB** в [App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237).

Незалежно від того, чи ви графічний дизайнер, який підбирає колір бренду зі зразка, розробник, що витягує коди для CSS, дизайнер інтер'єру, що бере зразки фарб, або любитель, що будує мудборд — Color Picker дає точні готові до копіювання значення одним дотиком. Усе оброблення відбувається локально на пристрої, без збору даних і без реклами.

## Ключові можливості

### Ідентифікація кольорів

- **Живий піпетка з камери** — Наведіть точне перехрестя на будь-яку реальну поверхню для захоплення точного кольору пікселя в реальному часі. Працює як піпетка для фізичного світу.
- **Піпетка з фото** — Витягайте кольори з будь-якого зображення у галереї. Торкніться будь-де на фото, щоб визначити його колір.
- **Декілька форматів** — Кожен захоплений колір відображається одночасно в HEX, RGB, HSL і CMYK, тож у вас завжди є потрібний формат.
- **Копіювання одним дотиком** — Торкніться будь-якого коду кольору, щоб миттєво скопіювати його в буфер, готовий для вставки у ваш дизайн-інструмент або редактор коду.

### Організація кольорів

- **Бібліотека і зразки** — Зберігайте улюблені кольори та організовуйте їх в особистій бібліотеці для швидкого доступу.
- **Конструктор палітр** — Створюйте власні палітри вручну або будуйте їх із захоплених кольорів. Комбінуйте споріднені кольори у цілісні палітри.
- **Поділитися та експорт** — Поділіться палітрами як текст, JSON, змінні CSS або зображення PNG. Експортуйте повну бібліотеку для резервного копіювання чи передачі колегам.

### Інструменти кольору

- **Перевірка контрасту** — Перевірте, чи два кольори відповідають вимогам контрасту для читабельності тексту.
- **Пропозиції гармоній** — Генеруйте компліментарні, тріадичні, аналогові та інші гармонійні комбінації з будь-якого вибраного кольору.
- **Відповідність RAL** — Знайдіть найближчу референцію фарби RAL для будь-якого захопленого кольору — корисно для фарбування, промислового дизайну та виробництва.

### Зручність використання

- **Чистий, інтуїтивний інтерфейс** з мінімалістичним дизайном, зосередженим на швидкому захопленні кольорів і легкій навігації між бібліотекою, палітрами та інструментами.
- **Працює офлайн** — Не потрібне інтернет-з'єднання. Усе оброблення кольору відбувається на вашому пристрої.
- **Темний режим** — Повна підтримка світлої та темної тем для комфортного використання в будь-якому середовищі.
- **Резервне копіювання та відновлення** — Експортуйте та імпортуйте всю бібліотеку кольорів і палітр як JSON через застосунок Файли.

## Як працює піпетка з камери

Color Picker використовує задню камеру телефону для захоплення живого відеопотоку. Точне перехрестя розміщене в центрі екрана, позначаючи точний піксель, що вибирається. Коли ви рухаєте камеру, застосунок читає значення пікселя під перехрестям у реальному часі та одночасно перетворює його на кілька форматів (HEX, RGB, HSL, CMYK).

Для вибору кольору з фото застосунок завантажує вибране зображення і дозволяє торкнутися будь-якого пікселя. Колір за координатами дотику витягується з даних зображення і перетворюється на всі підтримувані формати. Працює з будь-яким форматом зображення, що підтримується галереєю пристрою.

Усе оброблення виконується локально на вашому пристрої за допомогою обладнання камери та GPU. Жодні зображення чи дані кольору не надсилаються на сервер. Застосунок вимагає дозвіл камери лише для живого вибору кольору та доступ до галереї лише для витягування кольорів із фото.

**Наскільки точний піпетка з камери?** Точність залежить від сенсора камери та умов освітлення. Для найкращих результатів забезпечте добре нейтральне освітлення та тримайте пристрій стабільно. Застосунок захоплює точний колір пікселя з потоку камери, тож точність обмежена лише обладнанням.

**Чи можу я брати кольори з будь-якої фотографії?** Так, ви можете брати кольори з будь-якого зображення на пристрої — фото, скріншотів, завантажених зображень і мокапів дизайну.

## Часті питання

**П: Чи Color Picker безкоштовний?**
В: Так, застосунок повністю безкоштовний на Android та iOS. Без реклами та без покупок у застосунку.

**П: Чи працює без інтернету?**
В: Так, Color Picker працює повністю офлайн. Уся ідентифікація та оброблення кольорів відбувається локально на вашому пристрої.

**П: Які формати кольорів підтримує?**
В: Застосунок підтримує HEX (напр. #FF5733), RGB (напр. 255, 87, 51), HSL (напр. 11°, 100%, 60%) та CMYK (напр. 0%, 66%, 80%, 0%). Усі формати відображаються одночасно для кожного захопленого кольору.

**П: Чи можу я використовувати це як піпетку?**
В: Так, Color Picker працює точно як піпетка з настільного дизайн-програмного забезпечення, але для реального світу. Наведіть камеру на будь-яку поверхню, і перехрестя визначить точний колір пікселя.

**П: Чи збирає мої дані?**
В: Ні. Color Picker не збирає жодних даних користувача. Усі ваші кольори, палітри та налаштування зберігаються лише на вашому пристрої. Жодної аналітики, жодного відстеження, жодної реклами.

**П: Чи можу я експортувати палітри для дизайн-інструментів?**
В: Так, ви можете експортувати палітри як JSON, змінні CSS, текстові описи або зображення PNG. Легко ділитися з колегами або імпортувати у Figma, Sketch, VS Code або будь-який інший інструмент.

**П: Що таке відповідність RAL?**
В: RAL — це стандартизована система кольорів, що використовується переважно у фарбуванні, покриттях і промисловому дизайні. Color Picker знаходить найближчу референцію RAL для будь-якого захопленого кольору.

**П: Які пристрої підтримуються?**
В: Android 5.0 (Lollipop) та вище, і iOS 13.0 та вище. Застосунок також працює на iPad, iPod touch, Mac з Apple Silicon і Apple Vision.

**П: Яка різниця між версіями Android та iOS?**
В: Обидві версії пропонують однакову основну функціональність. Android-версія називається «Color Picker: Camera HEX RGB», а iOS — «Color Picker Camera: HEX RGB».
