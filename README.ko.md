## Color Picker: 카메라 HEX RGB란?

**Color Picker: 카메라 HEX RGB**는 [lapnito.cz s.r.o.](https://lapnito.cz)가 개발한 무료 색상 식별 앱으로, 휴대폰 카메라나 갤러리의 사진을 사용하여 현실 세계에서 정확한 색상을 캡처할 수 있습니다. 십자선을 어떤 표면에든 향하게 하면 앱이 즉시 픽셀 색상을 식별하고 HEX, RGB, HSL, CMYK 형식으로 값을 제공합니다.

[Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker)에서는 **Color Picker: Camera HEX RGB**로, [App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237)에서는 **Color Picker Camera: HEX RGB**로 제공됩니다.

물리적 샘플에서 브랜드 색상을 일치시키는 그래픽 디자이너, CSS용 정확한 색상 코드를 추출하는 개발자, 벽 페인트를 샘플링하는 인테리어 데코레이터, 무드보드를 만드는 애호가 — Color Picker는 한 번의 탭으로 정확하고 복사 가능한 색상 값을 제공합니다. 모든 처리는 기기에서 로컬로 이루어지며 데이터 수집 제로, 광고 없음.

## 주요 기능

### 색상 식별

- **실시간 카메라 컬러 피커** — 정밀한 십자선을 현실 세계의 어떤 표면에든 향하게 하여 실시간으로 정확한 픽셀 색상을 캡처합니다. 물리적 세계의 스포이드 도구처럼 작동합니다.
- **사진 컬러 피커** — 사진 갤러리의 어떤 이미지에서도 색상을 추출합니다. 사진의 어느 곳이든 탭하여 색상을 식별합니다.
- **다양한 색상 형식** — 캡처된 모든 색상은 HEX, RGB, HSL, CMYK로 동시에 표시되므로 필요한 형식이 항상 준비되어 있습니다.
- **원탭 복사** — 어떤 색상 코드든 탭하면 즉시 클립보드에 복사되어 디자인 도구나 코드 편집기에 붙여넣을 준비가 됩니다.

### 색상 구성

- **색상 라이브러리 및 스와치** — 좋아하는 색상을 저장하고 나중에 빠르게 참조할 수 있도록 개인 라이브러리에 구성합니다.
- **팔레트 빌더** — 사용자 정의 색상 팔레트를 수동으로 만들거나 선택한 색상에서 구축합니다. 관련 색상을 일관된 팔레트로 결합합니다.
- **공유 및 내보내기** — 팔레트를 텍스트, JSON, CSS 변수 또는 PNG 이미지로 공유합니다. 백업 또는 팀원에게 전달하기 위해 전체 라이브러리를 내보냅니다.

### 색상 도구

- **명도 대비 검사기** — 두 색상이 텍스트 가독성을 위한 대비 접근성 요구사항을 충족하는지 확인합니다.
- **색상 조화 제안** — 선택한 색상에서 보색, 삼색, 유사색 및 기타 조화로운 색상 조합을 생성합니다.
- **RAL 색상 일치** — 캡처된 색상에 가장 가까운 RAL 페인트 참조를 찾습니다 — 페인팅, 산업 디자인, 제조에 유용합니다.

### 사용자 경험

- **깨끗하고 직관적인 인터페이스** — 빠른 색상 캡처와 라이브러리, 팔레트, 도구 간의 쉬운 탐색에 초점을 맞춘 미니멀 디자인.
- **오프라인 작동** — 인터넷 연결이 필요하지 않습니다. 모든 색상 처리는 기기에서 완전히 이루어집니다.
- **다크 모드 지원** — 모든 환경에서 편안한 사용을 위한 완전한 라이트 및 다크 테마 지원.
- **백업 및 복원** — 파일 앱을 통해 전체 색상 라이브러리와 팔레트를 JSON으로 내보내고 가져옵니다.

| **교육 및 색상 이론** | 색상 조화, 보색 쌍 및 대비 비율을 탐구합니다. 다양한 색상 형식(HEX, RGB, HSL, CMYK)이 서로 어떻게 관련되어 있는지 배웁니다. |
| **접근성 테스트** | 디자인이 WCAG 접근성 지침을 충족하는지 확인하기 위해 전경과 배경 색상 간의 대비 비율을 확인합니다. |

## 카메라 색상 선택의 작동 방식

Color Picker는 휴대폰의 후면 카메라를 사용하여 실시간 비디오 피드를 캡처합니다. 정밀한 십자선 오버레이가 화면 중앙에 배치되어 샘플링되는 정확한 픽셀을 표시합니다. 카메라를 움직이면 앱이 십자선 아래 픽셀의 색상 값을 실시간으로 읽고 여러 색상 형식(HEX, RGB, HSL, CMYK)으로 동시에 변환합니다.

사진 기반 색상 선택의 경우 앱은 선택한 이미지를 로드하고 어떤 픽셀이든 탭할 수 있도록 합니다. 탭 좌표의 색상은 이미지의 픽셀 데이터에서 추출되어 지원되는 모든 형식으로 변환됩니다. 기기의 사진 라이브러리에서 지원하는 모든 이미지 형식에서 작동합니다.

모든 색상 처리는 카메라 하드웨어와 GPU를 사용하여 기기에서 로컬로 수행됩니다. 어떤 이미지나 색상 데이터도 서버로 전송되지 않습니다. 앱은 실시간 색상 선택을 위한 카메라 권한과 갤러리 기반 색상 추출을 위한 사진 라이브러리 접근 권한만 요구합니다.

**카메라 컬러 피커는 얼마나 정확한가요?** 정확도는 카메라 센서와 조명 조건에 따라 다릅니다. 최상의 결과를 위해 좋고 중성적인 조명을 확보하고 기기를 안정적으로 유지하세요. 앱은 카메라 피드에서 정확한 픽셀 색상을 캡처하므로 정확도는 카메라 하드웨어 자체에 의해서만 제한됩니다.

**모든 사진에서 색상을 선택할 수 있나요?** 네, 기기에 저장된 모든 이미지에서 색상을 선택할 수 있습니다 — 사진, 스크린샷, 다운로드한 이미지 및 디자인 목업 모두 작동합니다.

## 자주 묻는 질문

**Q: Color Picker는 무료인가요?**
A: 네, 앱은 Android와 iOS 모두에서 완전히 무료로 다운로드하여 사용할 수 있습니다. 광고 없음, 인앱 구매 없음.

**Q: 인터넷 없이 작동하나요?**
A: 네, Color Picker는 완전히 오프라인으로 작동합니다. 모든 색상 감지와 처리는 기기에서 로컬로 이루어집니다. 인터넷 연결이 필요 없습니다.

**Q: 어떤 색상 형식을 지원하나요?**
A: 앱은 HEX(예: #FF5733), RGB(예: 255, 87, 51), HSL(예: 11°, 100%, 60%), CMYK(예: 0%, 66%, 80%, 0%)를 지원합니다. 선택한 모든 색상에 대해 모든 형식이 동시에 표시됩니다.

**Q: 스포이드 도구로 사용할 수 있나요?**
A: 네, Color Picker는 데스크톱 디자인 소프트웨어의 스포이드 도구와 정확히 같은 방식으로 작동하지만 현실 세계를 위한 것입니다. 카메라를 어떤 표면에든 향하게 하면 십자선이 정확한 픽셀 색상을 식별합니다.

**Q: 내 데이터를 수집하나요?**
A: 아니요. Color Picker는 사용자 데이터를 수집하지 않습니다. 모든 색상, 팔레트 및 설정은 기기에만 저장됩니다. 분석 없음, 추적 없음, 광고 없음.

**Q: 디자인 도구에서 사용하기 위해 팔레트를 내보낼 수 있나요?**
A: 네, 팔레트를 JSON, CSS 변수, 텍스트 설명 또는 PNG 이미지로 내보낼 수 있습니다. 팀원과 색상을 공유하거나 Figma, Sketch, VS Code 또는 기타 도구로 가져오기가 쉬워집니다.

**Q: RAL 일치란 무엇인가요?**
A: RAL은 주로 페인팅, 코팅 및 산업 디자인에 사용되는 표준화된 색상 시스템입니다. Color Picker는 선택한 모든 색상에 대해 가장 가까운 RAL 색상 참조를 찾습니다.

**Q: 어떤 기기가 지원되나요?**
A: Android 5.0 (Lollipop) 이상과 iOS 13.0 이상. 앱은 iPad, iPod touch, Apple Silicon Mac 및 Apple Vision에서도 실행됩니다.

**Q: Android와 iOS 버전의 차이점은 무엇인가요?**
A: 두 버전 모두 동일한 핵심 기능을 제공합니다. Android 버전은 "Color Picker: Camera HEX RGB", iOS 버전은 "Color Picker Camera: HEX RGB"로 불립니다.
