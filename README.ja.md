## Color Picker: カメラ HEX RGB とは?

**Color Picker: カメラ HEX RGB** は [lapnito.cz s.r.o.](https://lapnito.cz) が開発した無料のカラー識別アプリで、スマートフォンのカメラまたはギャラリーの任意の写真を使って現実世界から正確な色をキャプチャできます。照準を任意の表面に合わせると、アプリは瞬時にピクセルの色を識別し、HEX、RGB、HSL、CMYK 形式で値を提供します。

[Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker) で **Color Picker: Camera HEX RGB**、[App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237) で **Color Picker Camera: HEX RGB** として提供されています。

物理サンプルからブランドカラーをマッチさせるグラフィックデザイナー、CSS 用に正確なカラーコードを抽出する開発者、壁のペイントをサンプリングするインテリアデコレーター、ムードボードを作る愛好家 — Color Picker はワンタップで正確でコピー可能なカラー値を提供します。すべての処理はデバイス上でローカルに行われ、データ収集ゼロ、広告なしです。

## 主な機能

### 色の識別

- **ライブカメラカラーピッカー** — 精密な照準を現実世界の任意の表面に合わせ、リアルタイムで正確なピクセルの色をキャプチャします。物理世界のスポイトツールとして機能します。
- **写真カラーピッカー** — 写真ギャラリーの任意の画像から色を抽出します。写真の任意の場所をタップして色を識別します。
- **複数のカラー形式** — キャプチャされた各色は HEX、RGB、HSL、CMYK で同時に表示されるので、必要な形式が常に手元にあります。
- **ワンタップコピー** — 任意のカラーコードをタップしてクリップボードに即座にコピーし、デザインツールやコードエディタに貼り付ける準備ができます。

### 色の整理

- **カラーライブラリとスウォッチ** — お気に入りの色を保存し、後で素早く参照できるように個人ライブラリに整理します。
- **パレットビルダー** — カスタムカラーパレットを手動で作成するか、選択した色から構築します。関連する色を結合して一貫性のあるパレットを作成します。
- **共有とエクスポート** — パレットをテキスト、JSON、CSS 変数、PNG 画像として共有します。バックアップやチームメンバーへの引き継ぎ用にライブラリ全体をエクスポートします。

### カラーツール

- **コントラストチェッカー** — 2 つの色がテキストの可読性のためのコントラストアクセシビリティ要件を満たしているかどうかを検証します。
- **色彩調和の提案** — 選択した色から補色、トライアド、アナロガス、その他の調和のとれた色の組み合わせを生成します。
- **RAL カラーマッチング** — キャプチャした色に最も近い RAL ペイント参照を見つけます — ペイント、工業デザイン、製造に便利です。

### ユーザー体験

- **クリーンで直感的なインターフェース** — 素早い色キャプチャとライブラリ、パレット、ツール間の簡単なナビゲーションに焦点を当てたミニマルデザイン。
- **オフラインで動作** — インターネット接続は不要です。すべての色処理は完全にデバイス上で行われます。
- **ダークモード対応** — 任意の環境で快適に使用できるライトテーマとダークテーマの完全サポート。
- **バックアップと復元** — ファイルアプリ経由でカラーライブラリとパレット全体を JSON としてエクスポートおよびインポートします。

## カメラカラーピッキングの仕組み

Color Picker は、スマートフォンのリアカメラを使用してライブビデオフィードをキャプチャします。精密な照準オーバーレイが画面の中央に配置され、サンプリングされている正確なピクセルをマークします。カメラを動かすと、アプリは照準下のピクセルのカラー値をリアルタイムで読み取り、複数のカラー形式（HEX、RGB、HSL、CMYK）に同時に変換します。

写真ベースのカラーピッキングでは、アプリが選択された画像を読み込み、任意のピクセルをタップできるようにします。タップ座標の色は画像のピクセルデータから抽出され、サポートされているすべての形式に変換されます。デバイスの写真ライブラリがサポートする任意の画像形式で動作します。

すべてのカラー処理は、カメラハードウェアと GPU を使用してデバイス上でローカルに実行されます。画像やカラーデータがサーバーに送信されることはありません。アプリはライブカラーピッキングのためのみにカメラ権限を必要とし、ギャラリーベースのカラー抽出のためのみに写真ライブラリアクセスを必要とします。

**カメラカラーピッカーはどの程度正確ですか?** 精度はカメラセンサーと照明条件に依存します。最良の結果を得るには、良好でニュートラルな照明を確保し、デバイスを安定させて保持してください。アプリはカメラフィードから正確なピクセルの色をキャプチャするため、精度はカメラハードウェア自体によってのみ制限されます。

**任意の写真から色を選択できますか?** はい、デバイスに保存されている任意の画像から色を選択できます — 写真、スクリーンショット、ダウンロードされた画像、デザインモックアップすべて動作します。

## よくある質問

**Q: Color Picker は無料ですか?**
A: はい、アプリは Android と iOS の両方で完全に無料でダウンロードして使用できます。広告もアプリ内課金もありません。

**Q: インターネットなしで動作しますか?**
A: はい、Color Picker は完全にオフラインで動作します。すべての色検出と処理はデバイス上でローカルに行われます。インターネット接続は必要ありません。

**Q: どのカラー形式をサポートしていますか?**
A: アプリは HEX（例: #FF5733）、RGB（例: 255、87、51）、HSL（例: 11°、100%、60%）、CMYK（例: 0%、66%、80%、0%）をサポートしています。選択した各色ですべての形式が同時に表示されます。

**Q: スポイトツールとして使えますか?**
A: はい、Color Picker はデスクトップデザインソフトウェアのスポイトツールとまったく同じように動作しますが、現実世界用です。カメラを任意の表面に向けると、照準が正確なピクセルの色を識別します。

**Q: データを収集しますか?**
A: いいえ。Color Picker はユーザーデータを収集しません。すべての色、パレット、設定はデバイスにのみ保存されます。分析なし、追跡なし、広告なし。

**Q: デザインツールで使用するためにパレットをエクスポートできますか?**
A: はい、パレットを JSON、CSS 変数、テキスト記述、PNG 画像としてエクスポートできます。これにより、チームメンバーと色を共有したり、Figma、Sketch、VS Code、その他のツールにインポートしたりすることが簡単になります。

**Q: RAL マッチングとは何ですか?**
A: RAL は主にペイント、コーティング、工業デザインで使用される標準化されたカラーシステムです。Color Picker は選択した色に最も近い RAL カラー参照を見つけます。

**Q: どのデバイスがサポートされていますか?**
A: Android 5.0 (Lollipop) 以降と iOS 13.0 以降。アプリは iPad、iPod touch、Apple Silicon 搭載 Mac、Apple Vision でも動作します。

**Q: Android と iOS バージョンの違いは?**
A: 両バージョンとも同じ中核機能を提供します。Android バージョンは「Color Picker: Camera HEX RGB」、iOS バージョンは「Color Picker Camera: HEX RGB」と呼ばれます。
