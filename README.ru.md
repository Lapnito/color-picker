## Что такое Color Picker: Камера HEX RGB?

**Color Picker: Камера HEX RGB** — это бесплатное приложение для определения цветов от [lapnito.cz s.r.o.](https://lapnito.cz), которое позволяет захватывать точные цвета из реального мира с помощью камеры телефона или любой фотографии из галереи. Наведите прицел на любую поверхность, и приложение мгновенно определит цвет пикселя и предоставит его значение в форматах HEX, RGB, HSL и CMYK.

Доступно как **Color Picker: Camera HEX RGB** в [Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker) и как **Color Picker Camera: HEX RGB** в [App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237).

Будь вы графическим дизайнером, подбирающим цвет бренда с физического образца, разработчиком, извлекающим точные коды цветов для CSS, декоратором интерьеров, берущим образцы красок для стен, или любителем, создающим мудборд — Color Picker даёт вам точные, готовые к копированию значения цветов одним касанием. Вся обработка происходит локально на устройстве, без сбора данных и без рекламы.

## Ключевые возможности

### Определение цветов

- **Живая пипетка камеры** — Наведите точный прицел на любую поверхность реального мира, чтобы захватить точный цвет пикселя в реальном времени. Работает как пипетка для физического мира.
- **Пипетка из фото** — Извлекайте цвета из любого изображения вашей галереи. Коснитесь в любом месте фотографии, чтобы определить её цвет.
- **Множество форматов цвета** — Каждый захваченный цвет отображается одновременно в HEX, RGB, HSL и CMYK, поэтому у вас всегда есть нужный формат.
- **Копирование одним касанием** — Коснитесь любого кода цвета, чтобы мгновенно скопировать его в буфер обмена, готовый для вставки в ваш инструмент дизайна или редактор кода.

### Организация цветов

- **Библиотека цветов и образцы** — Сохраняйте любимые цвета и организуйте их в личной библиотеке для быстрого доступа.
- **Создатель палитр** — Создавайте пользовательские палитры вручную или стройте их из захваченных цветов. Комбинируйте связанные цвета в согласованные палитры.
- **Поделиться и экспорт** — Делитесь палитрами в виде текста, JSON, переменных CSS или изображений PNG. Экспортируйте всю библиотеку для резервного копирования или передачи коллегам.

### Инструменты цвета

- **Проверка контраста** — Проверьте, соответствуют ли два цвета требованиям контраста для читаемости текста.
- **Предложения гармоний** — Генерируйте комплементарные, триадические, аналоговые и другие гармоничные комбинации из любого выбранного цвета.
- **Соответствие RAL** — Найдите ближайшую справочную краску RAL для любого захваченного цвета — полезно для покраски, промышленного дизайна и производства.

## Как работает пипетка камеры

Color Picker использует заднюю камеру телефона для захвата живого видеопотока. Точный прицел расположен в центре экрана, отмечая точный пиксель, который берётся в качестве образца. При движении камеры приложение считывает значение пикселя под прицелом в реальном времени и одновременно преобразует его в несколько форматов (HEX, RGB, HSL, CMYK).

Для выбора цвета из фото приложение загружает выбранное изображение и позволяет коснуться любого пикселя. Цвет в координатах касания извлекается из данных изображения и преобразуется во все поддерживаемые форматы. Работает с любым форматом изображения, поддерживаемым галереей устройства.

Вся обработка выполняется локально на устройстве с использованием оборудования камеры и GPU. Никакие изображения или данные цвета не отправляются на сервер. Приложение требует разрешение камеры только для живой пипетки и доступ к галерее только для извлечения цветов из фотографий.

**Насколько точна пипетка камеры?** Точность зависит от сенсора камеры и условий освещения. Для лучших результатов обеспечьте хорошее нейтральное освещение и держите устройство стабильно. Приложение захватывает точный цвет пикселя из потока камеры, поэтому точность ограничена только оборудованием.

**Могу ли я извлекать цвета из любой фотографии?** Да, вы можете извлекать цвета из любого изображения на устройстве — фотографий, скриншотов, загруженных изображений и макетов дизайна.

## Часто задаваемые вопросы

**В: Color Picker бесплатный?**
О: Да, приложение полностью бесплатное на Android и iOS. Без рекламы и без покупок в приложении.

**В: Работает ли без интернета?**
О: Да, Color Picker работает полностью офлайн. Всё обнаружение и обработка цветов происходит локально на вашем устройстве.

**В: Какие форматы цвета поддерживает?**
О: Приложение поддерживает HEX (напр. #FF5733), RGB (напр. 255, 87, 51), HSL (напр. 11°, 100%, 60%) и CMYK (напр. 0%, 66%, 80%, 0%). Все форматы отображаются одновременно для каждого захваченного цвета.

**В: Могу ли я использовать это как пипетку?**
О: Да, Color Picker работает точно как пипетка из настольного дизайн-ПО, но для реального мира. Наведите камеру на любую поверхность, и прицел определит точный цвет пикселя.

**В: Собирает ли приложение мои данные?**
О: Нет. Color Picker не собирает никаких пользовательских данных. Все ваши цвета, палитры и настройки хранятся только на вашем устройстве. Никакой аналитики, никакого отслеживания, никакой рекламы.

**В: Могу ли я экспортировать палитры для дизайн-инструментов?**
О: Да, вы можете экспортировать палитры в виде JSON, переменных CSS, текстовых описаний или изображений PNG. Легко делиться с коллегами или импортировать в Figma, Sketch, VS Code или любой другой инструмент.

**В: Что такое соответствие RAL?**
О: RAL — это стандартизированная система цветов, используемая в основном в покраске, покрытиях и промышленном дизайне. Color Picker находит ближайшую справочную краску RAL для любого захваченного цвета.

**В: Какие устройства поддерживаются?**
О: Android 5.0 (Lollipop) и выше, iOS 13.0 и выше. Приложение также работает на iPad, iPod touch, Mac с Apple Silicon и Apple Vision.

**В: В чём разница между версиями Android и iOS?**
О: Обе версии предлагают одинаковую основную функциональность. Версия Android называется «Color Picker: Camera HEX RGB», а iOS — «Color Picker Camera: HEX RGB».
