## Color Picker: กล้อง HEX RGB คืออะไร?

**Color Picker: กล้อง HEX RGB** เป็นแอปพลิเคชันระบุสีฟรีจาก [lapnito.cz s.r.o.](https://lapnito.cz) ที่ช่วยให้คุณจับสีที่แม่นยำจากโลกจริงโดยใช้กล้องโทรศัพท์หรือภาพถ่ายใด ๆ จากแกลเลอรี เล็งเป้าเล็งไปที่พื้นผิวใด ๆ และแอปจะระบุสีพิกเซลทันทีและให้ค่าในรูปแบบ HEX, RGB, HSL และ CMYK

มีให้ใช้งานในชื่อ **Color Picker: Camera HEX RGB** บน [Google Play](https://play.google.com/store/apps/details?id=com.tomas.color_picker) และในชื่อ **Color Picker Camera: HEX RGB** บน [App Store](https://apps.apple.com/us/app/color-picker-camera-hex-rgb/id6759371237)

ไม่ว่าคุณจะเป็นนักออกแบบกราฟิกที่จับคู่สีแบรนด์จากตัวอย่างจริง นักพัฒนาที่สกัดรหัสสีที่แม่นยำสำหรับ CSS นักตกแต่งภายในที่เก็บตัวอย่างสีผนัง หรือผู้ที่ชื่นชอบสร้างมู้ดบอร์ด — Color Picker ให้ค่าสีที่แม่นยำพร้อมคัดลอกด้วยการแตะเพียงครั้งเดียว การประมวลผลทั้งหมดเกิดขึ้นในอุปกรณ์ของคุณ ไม่มีการเก็บข้อมูลและไม่มีโฆษณา

## คุณสมบัติหลัก

### การระบุสี

- **ตัวเลือกสีกล้องสด** — เล็งเป้าเล็งที่แม่นยำไปที่พื้นผิวโลกจริงใด ๆ เพื่อจับสีพิกเซลที่แม่นยำแบบเรียลไทม์ ทำงานเหมือนเครื่องมือหลอดหยดสำหรับโลกกายภาพ
- **ตัวเลือกสีจากรูป** — สกัดสีจากภาพใด ๆ ในแกลเลอรีรูปภาพของคุณ แตะที่ใดก็ได้บนภาพเพื่อระบุสีของมัน
- **รูปแบบสีหลากหลาย** — ทุกสีที่จับได้จะแสดงพร้อมกันใน HEX, RGB, HSL และ CMYK ดังนั้นคุณจะมีรูปแบบที่ต้องการเสมอ
- **คัดลอกด้วยการแตะเดียว** — แตะที่รหัสสีใด ๆ เพื่อคัดลอกไปยังคลิปบอร์ดทันที พร้อมที่จะวางในเครื่องมือออกแบบหรือโปรแกรมแก้ไขโค้ดของคุณ

### การจัดระเบียบสี

- **ไลบรารีสีและตัวอย่างสี** — บันทึกสีโปรดของคุณและจัดระเบียบในไลบรารีส่วนตัวสำหรับอ้างอิงอย่างรวดเร็ว
- **ตัวสร้างพาเลต** — สร้างพาเลตสีแบบกำหนดเองด้วยตนเองหรือสร้างจากสีที่เลือก รวมสีที่เกี่ยวข้องเป็นพาเลตที่สอดคล้องกัน
- **การแบ่งปันและส่งออก** — แบ่งปันพาเลตเป็นข้อความ JSON, ตัวแปร CSS หรือภาพ PNG ส่งออกไลบรารีทั้งหมดของคุณสำหรับการสำรองข้อมูลหรือมอบให้เพื่อนร่วมทีม

### เครื่องมือสี

- **ตัวตรวจสอบคอนทราสต์** — ตรวจสอบว่าสองสีตรงตามข้อกำหนดคอนทราสต์การเข้าถึงสำหรับการอ่านข้อความหรือไม่
- **คำแนะนำความสมดุลของสี** — สร้างการผสมสีเสริม สามสี คล้ายคลึง และความสมดุลอื่น ๆ จากสีที่เลือกใด ๆ
- **การจับคู่สี RAL** — ค้นหาอ้างอิงสี RAL ที่ใกล้ที่สุดสำหรับสีใด ๆ ที่จับได้ — มีประโยชน์สำหรับการทาสี การออกแบบอุตสาหกรรม และการผลิต

### ประสบการณ์ผู้ใช้

- **อินเทอร์เฟซที่สะอาดและใช้งานง่าย** ด้วยการออกแบบที่เรียบง่ายเน้นการจับสีที่รวดเร็วและการนำทางที่ง่ายระหว่างไลบรารี พาเลต และเครื่องมือ
- **ทำงานแบบออฟไลน์** — ไม่ต้องมีการเชื่อมต่ออินเทอร์เน็ต การประมวลผลสีทั้งหมดเกิดขึ้นโดยสมบูรณ์บนอุปกรณ์ของคุณ
- **รองรับโหมดมืด** — รองรับธีมสว่างและมืดเต็มรูปแบบเพื่อการใช้งานที่สบายในทุกสภาพแวดล้อม
- **สำรองและคืนค่า** — ส่งออกและนำเข้าไลบรารีสีและพาเลตทั้งหมดเป็น JSON ผ่านแอปไฟล์

## วิธีการทำงานของการเลือกสีด้วยกล้อง

Color Picker ใช้กล้องด้านหลังของโทรศัพท์เพื่อจับสตรีมวิดีโอสด เป้าเล็งแม่นยำวางไว้ที่กึ่งกลางหน้าจอ ทำเครื่องหมายพิกเซลที่แม่นยำที่กำลังถูกสุ่มตัวอย่าง เมื่อคุณขยับกล้อง แอปจะอ่านค่าสีของพิกเซลใต้เป้าเล็งแบบเรียลไทม์และแปลงเป็นรูปแบบสีหลายรูปแบบ (HEX, RGB, HSL, CMYK) พร้อมกัน

สำหรับการเลือกสีตามภาพ แอปจะโหลดภาพที่เลือกและให้คุณแตะที่พิกเซลใด ๆ สีที่พิกัดแตะถูกสกัดจากข้อมูลพิกเซลของภาพและแปลงเป็นรูปแบบที่รองรับทั้งหมด ใช้งานได้กับรูปแบบภาพใด ๆ ที่ไลบรารีภาพถ่ายของอุปกรณ์คุณรองรับ

การประมวลผลสีทั้งหมดดำเนินการในเครื่องบนอุปกรณ์ของคุณโดยใช้ฮาร์ดแวร์กล้องและ GPU ไม่มีการส่งภาพหรือข้อมูลสีไปยังเซิร์ฟเวอร์ใด ๆ แอปต้องการสิทธิ์กล้องเฉพาะสำหรับการเลือกสีสดและการเข้าถึงไลบรารีภาพถ่ายเฉพาะสำหรับการสกัดสีตามแกลเลอรี

**ตัวเลือกสีกล้องแม่นยำแค่ไหน?** ความแม่นยำขึ้นอยู่กับเซ็นเซอร์กล้องและสภาพแสงของคุณ เพื่อผลลัพธ์ที่ดีที่สุด ให้แสงสว่างที่ดีและเป็นกลางและถืออุปกรณ์ให้นิ่ง แอปจับสีพิกเซลที่แม่นยำจากสตรีมกล้อง ดังนั้นความแม่นยำจึงถูกจำกัดเฉพาะฮาร์ดแวร์กล้องเองเท่านั้น

**ฉันสามารถเลือกสีจากภาพใด ๆ ได้ไหม?** ใช่ คุณสามารถเลือกสีจากภาพใด ๆ ที่เก็บไว้ในอุปกรณ์ของคุณ — ภาพถ่าย ภาพหน้าจอ ภาพที่ดาวน์โหลด และโมคอัพการออกแบบทั้งหมดใช้งานได้

## คำถามที่พบบ่อย

**ถ: Color Picker ฟรีไหม?**
ต: ใช่ แอปฟรีโดยสมบูรณ์สำหรับดาวน์โหลดและใช้งานบน Android และ iOS ไม่มีโฆษณาและไม่มีการซื้อในแอป

**ถ: ทำงานโดยไม่มีอินเทอร์เน็ตหรือไม่?**
ต: ใช่ Color Picker ทำงานแบบออฟไลน์โดยสมบูรณ์ การตรวจจับและประมวลผลสีทั้งหมดเกิดขึ้นในเครื่องบนอุปกรณ์ของคุณ ไม่จำเป็นต้องมีการเชื่อมต่ออินเทอร์เน็ต

**ถ: รองรับรูปแบบสีอะไรบ้าง?**
ต: แอปรองรับ HEX (เช่น #FF5733), RGB (เช่น 255, 87, 51), HSL (เช่น 11°, 100%, 60%) และ CMYK (เช่น 0%, 66%, 80%, 0%) รูปแบบทั้งหมดแสดงพร้อมกันสำหรับทุกสีที่เลือก

**ถ: ฉันสามารถใช้เป็นเครื่องมือหลอดหยดได้ไหม?**
ต: ใช่ Color Picker ทำงานเหมือนเครื่องมือหลอดหยดจากซอฟต์แวร์ออกแบบเดสก์ท็อปทุกประการ แต่สำหรับโลกจริง เล็งกล้องไปที่พื้นผิวใด ๆ และเป้าเล็งจะระบุสีพิกเซลที่แม่นยำ

**ถ: มันเก็บข้อมูลของฉันหรือไม่?**
ต: ไม่ Color Picker ไม่เก็บข้อมูลผู้ใช้ใด ๆ สีทั้งหมด พาเลต และการตั้งค่าของคุณถูกเก็บเฉพาะในอุปกรณ์ของคุณ ไม่มีการวิเคราะห์ ไม่มีการติดตาม และไม่มีโฆษณา

**ถ: ฉันสามารถส่งออกพาเลตของฉันเพื่อใช้ในเครื่องมือออกแบบได้ไหม?**
ต: ใช่ คุณสามารถส่งออกพาเลตเป็น JSON, ตัวแปร CSS, คำอธิบายข้อความ หรือภาพ PNG สิ่งนี้ทำให้ง่ายต่อการแบ่งปันสีกับเพื่อนร่วมทีมหรือนำเข้าไปยัง Figma, Sketch, VS Code หรือเครื่องมืออื่น ๆ

**ถ: การจับคู่ RAL คืออะไร?**
ต: RAL เป็นระบบสีมาตรฐานที่ใช้เป็นหลักในการทาสี การเคลือบ และการออกแบบอุตสาหกรรม Color Picker หาอ้างอิงสี RAL ที่ใกล้ที่สุดสำหรับสีที่เลือกใด ๆ

**ถ: อุปกรณ์ใดได้รับการสนับสนุน?**
ต: Android 5.0 (Lollipop) ขึ้นไป และ iOS 13.0 ขึ้นไป แอปยังทำงานบน iPad, iPod touch, Mac ที่มี Apple Silicon และ Apple Vision

**ถ: ความแตกต่างระหว่างเวอร์ชัน Android และ iOS คืออะไร?**
ต: ทั้งสองเวอร์ชันมีฟังก์ชันหลักเดียวกัน เวอร์ชัน Android เรียกว่า "Color Picker: Camera HEX RGB" และเวอร์ชัน iOS เรียกว่า "Color Picker Camera: HEX RGB"
